Minecraft Server Connection:

Step 1: Launch Minecraft: Java Edition and navigate to the "Multiplayer" page

Step 2: Click the "Add Server" Button in the bottom right of the menu.

Step 3: In the "Server Address" bar, type the Server IP Address as listed above. (You can set the Server Name to whatever you would like)

Step 4: Click "Done" and double-click on the newly created server to connect.
